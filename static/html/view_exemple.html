Réalisation du Backend d'une application mobile de gestion de brocantes

 

Bonjour à toutes et à tous,

Nous vous proposons aujourd'hui une étude de réalisation du backend d'une application mobile. L'application existe déjà, mais possède un code comportant trop de problèmes. L'objectif serait donc de refaire l'application. C'est une application de gestion de brocante.

DOMAINE

Application Mobile

RÉTRIBUTION

3400€-4000€

DIFFICULTÉ

Elevée

Compétences

Nous recherchons un·e intervenant·e ayant de bonnes compétences en backend et familier de la base de données Mango DB. L'intervenant doit également être capable de réaliser un déploiement sur IOS et Android. Enfin, il sera amené à travailler en équipe.

Échéances

Le client désire commencer le plus tôt possible. La mission devrait durer plusieurs mois.

Description

L'application est une application de gestion de brocante. Elle devra comporter les fonctionnalités suivantes : création et suppression de comptes et de brocantes, la localisation des brocantes sur une carte, la possibilité de prendre des photos, la possibilité d'utiliser un calendrier pour définir la date de la brocantes, une fonctionnalité de messagerie, de partage de l'application via sms et mail, la possibilité de passer d'un compte organisateur à participant, la possibilité de payer dans l'application (actuellement, cela se fait via Stripe), ainsi que la possibilité de changer la langue.
